Scope
Scope matlab ek variable ki reach kahatak hai ya ek variable kaha kaha se access kiya ja sakta hai
1. Function Scope
Function ke andar hi use ho sakti hai value usko function scope bolte hai

2. Global Scope
Global scope matlab poore code mai kahase bhi valueko access kar sakte hai usko Global scope bolte hai

3. Block Scope
BLock scope matlab {} curly brace mein use ho sakte hai isme function word ka use nhi kiya jata
E.g.: if loop, if-else loop, etc. ye sab block scope mai ate hai

function clickLimitor(){
    let click = 0;          --> this is a pvt var this var can't
                                be changed outside this closure function even global var can't change the value of this var
    return function(){
        if(click<5){
            click++;
            console.log(`clicked: ${click} time`);
        }
        else{
            console.error(`limit exceeded`);
        }
    }
}

let fnccl = clickLimimtor();
fnccl();        --> clicked: 1 time
fnccl();        --> clicked: 2 time
